<h2 id="projects">项目经历</h2>

+ **催促熊（WeChat Mini Program）**

  基于Python设计并开发了一款面向拖延问题的效率工具小程序，通过**任务拆解机制**将复杂目标分解为多个可执行步骤，帮助用户逐步推进任务并降低行动门槛。

  + 设计 **任务拆解系统**，支持将复杂目标分解为步骤化任务并按顺序执行  
  + 实现 **任务进度管理与状态记录**，提升用户任务完成的可追踪性  
  + 使用 **微信小程序原生框架** 完成前端界面与交互逻辑开发  
  + 基于 **云数据库** 实现用户任务数据存储与管理  
  + 小程序上线后累计用户 **3000+**

* **FPS-King（双人联机狙击游戏）**

  基于 **Rust 与 Bevy ECS 游戏引擎** 设计并开发了一款双人对战狙击游戏，支持本地双人模式与局域网联机模式，围绕“部分可见性”核心机制，实现复杂视线检测、实时渲染与网络同步，探索游戏系统设计与实时交互实现。

  * 基于 **Bevy ECS（Entity-Component-System）架构** 实现玩家控制、碰撞检测、回合逻辑与游戏状态管理，提升系统模块化与可维护性
  * 设计并实现 **部分可见性系统**，结合射线检测与可破坏墙体机制，实现“仅能透过破碎区域观察敌方”的动态视野遮挡效果
  * 使用 **RenderLayers 分层渲染机制** 实现第一人称狙击视角与 2.5D 俯视视角的双视角分屏同步渲染
  * 设计 **可破坏墙体系统**，支持砖块级独立破坏与状态更新，并实时影响视线检测与场景可见性
  * 基于 **UDP 通信与 ZeroTier 虚拟局域网** 实现多人联机与实时状态同步，同步玩家位置、射击事件及墙体破坏状态
  * 优化 **角色切换与相机管理机制**，通过缓冲与状态同步策略解决网络模式下的渲染竞态与相机切换问题
  * 实现 **身体部位差异化伤害判定** 与技能冷却机制，增强游戏策略性与交互体验


+ **Piano Simulation（钢琴声学仿真系统）**

  基于 **Rust** 设计并开发了一套面向钢琴物理建模与计算声学研究的声音合成系统，通过结合 **物理建模与采样合成技术**，实现具有动态响应与非谐性泛音结构的虚拟钢琴声学仿真。

  + 基于 **耗散刚性弦模型** 构建钢琴弦振动方程，模拟张力、刚度及频率相关阻尼等物理特性
  + 使用 **有限差分时域法（FDTD）** 对钢琴弦振动进行数值求解，实现钢琴击弦过程的动态仿真
  + 引入 **采样合成机制** 补偿纯物理建模在高频瞬态与音色细节上的不足，提升整体听感真实性
  + 基于 **rayon 并行计算框架** 实现多音符实时并行更新，提高系统实时性与计算效率
  + 使用 **rodio 音频框架** 实现实时音频流输出，并支持计算机键盘实时演奏
  + 基于 **midly** 实现 MIDI 文件解析与自动演奏功能，支持钢琴曲自动播放与实验测试
  + 项目开发过程中重点关注数值稳定性与实时性能，对计算声学、物理建模与高性能系统设计进行了系统性实践
