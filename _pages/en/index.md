<h2 id="projects">Projects</h2>

+ **CucuBear – Task Decomposition Productivity Tool (WeChat Mini Program)**

  Designed and developed a productivity mini-program aimed at reducing procrastination by decomposing complex goals into smaller, actionable steps, enabling users to complete tasks progressively and lowering the threshold for action.

  + Designed a **task decomposition system** that breaks complex goals into step-by-step subtasks executed sequentially.
  + Implemented **task progress tracking and state management**, enhancing the traceability of user tasks.
  + Developed the frontend UI and interaction logic using the **WeChat Mini Program native framework**.
  + Utilized **cloud database services** for user task storage and management.
  + Accumulated **3000+ users** after the mini-program was officially launched.

+ **FPS-King (Two-Player Online Sniper Game)**

  Designed and developed a two-player competitive sniper game based on **Rust and the Bevy ECS game engine**, supporting local two-player and LAN multiplayer modes. Centered around a "partial visibility" core mechanic, the project implements complex line-of-sight detection, real-time rendering, and network synchronization to explore game system design and real-time interaction.

  * Implemented player control, collision detection, round logic, and game state management based on the **Bevy ECS (Entity-Component-System) architecture**, improving system modularity and maintainability.
  * Designed and implemented a **partial visibility system**, combining raycasting and destructible wall mechanics to create a dynamic vision occlusion effect where players can only observe the enemy through broken areas.
  * Utilized **RenderLayers mechanism** for dual-perspective split-screen synchronized rendering of a first-person sniper view and a 2.5D top-down view.
  * Designed a **destructible wall system**, supporting independent brick-level destruction and state updates, dynamically affecting line-of-sight detection and scene visibility.
  * Implemented multiplayer connectivity and real-time state synchronization based on **UDP communication and ZeroTier virtual LAN**, synchronizing player positions, shooting events, and wall destruction states.
  * Optimized **character switching and camera management**, using buffering and state synchronization strategies to resolve rendering race conditions and camera switching issues in network mode.
  * Implemented **differentiated body-part damage detection** and a skill cooldown mechanism, enhancing game strategy and interactive experience.

+ **Piano Simulation (Acoustic Piano Simulation System)**

  Designed and developed an acoustic simulation and sound synthesis system for piano physical modeling and computational acoustics using **Rust**. By combining **physical modeling and sampling synthesis technologies**, it achieves virtual piano acoustic simulation with dynamic response and inharmonic overtone structures.

  + Constructed the piano string vibration equation based on the **dissipative stiff string model**, simulating physical characteristics such as tension, stiffness, and frequency-dependent damping.
  + Used the **Finite Difference Time Domain (FDTD) method** to numerically solve the string vibration, achieving dynamic simulation of the piano striking process.
  + Introduced a **sampling synthesis mechanism** to compensate for the shortcomings of pure physical modeling in high-frequency transients and tonal details, improving overall acoustic realism.
  + Implemented real-time parallel updates for multiple notes based on the **rayon parallel computing framework**, improving system real-time performance and computational efficiency.
  + Used the **rodio audio framework** for real-time audio stream output, supporting live performance via a computer keyboard.
  + Implemented MIDI file parsing and automatic performance functions using **midly**, supporting automatic piano playback and experimental testing.
  + Focused on numerical stability and real-time performance during development, conducting systematic practice in computational acoustics, physical modeling, and high-performance system design.
